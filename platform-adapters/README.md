# Platform Adapters

This folder contains template rule files used by `scripts/install_platform_adapters.py`.

- `cursor/fastmcp-mastery.mdc`: Cursor rule template
- `windsurf/fastmcp-mastery.md`: Windsurf rule template
- `cline/fastmcp-mastery.md`: Cline `.clinerules` block template

The installer replaces `__SKILL_PATH__` with `.agents/skills/fastmcp-mastery`.
